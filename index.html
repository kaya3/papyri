Papyri

Papyri is a programmable markup language, which can be compiled to HTML.

Papyri has a minimal syntax for common inline and block-level elements, and is particularly suitable for mixing text with source code and code listings. Documents written in Papyri are also fully programmable, as the language allows user-defined functions in Python 3.

You can click here to see the Papyri source for this page as an example.

The Papyri language and standard library is not yet stable enough for public release. When ready, the compiler will be published under an open-source license.
